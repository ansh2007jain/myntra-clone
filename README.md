# Myntra Clone

A front-end clone of the Myntra website built using HTML and CSS. This project was created to improve my front-end development skills by recreating the design and layout of a real-world e-commerce website.

## Features

- Responsive navigation bar
- Search bar
- Hero banner
- Shopping categories
- Footer section
- Clean and modern user interface

## Technologies Used

- HTML5
- CSS3

## Learning Outcomes

Through this project, I practiced:

- HTML page structure
- CSS Flexbox
- CSS positioning
- Responsive design basics
- Building a real-world website layout
